PRESENTATION 
Fonctionnement du site web : En effet j'utilise un filtre pour filtrer les infromations présentes dans mon fichier json , ensuite je fais le lien entre le fichier json et mes pages html.
Mon filtre permet de lier les reponses aux questionnaires aux informations présents dans mon fichier json et ainsi vous donne la reponse et l'ordinateur approprié pour vous en première position tout en vous présentant les autres ordinateurs qui pourraient vous interessez.
Pour les photos j'ai utilisé des APIs et des liens qui vont directement puiser ses photos sur Amazon ou sur Google.

J'ai ajouté la notion de super utilisateur qui permet d'ajouter des éléments dans le site web. 

Quand vous cliquez sur une photo d'ordinateur qui vous interesse, vous serez redirigé sur Amazon.


Consignes : Pour lancer le site web il vous faut d'abord executer les lignes de commandes suivantes sur votre terminal prompt ou sur vs code, éviter powershell.


Directives : 
installer d'abord angular sur votre ordinateur.
Ensuite lancer : npm install "pour installer les assets angular", sans ça vous aurez une erreur en lançant directement ng serve.
entrez la commande : ng serve "pour lancer le server angular"
entrez la commande : npm install -g json-server "pour lancer le server json"
entrez la commande : json-server data.json "permet de lancer mon fichier json et n'oublier pas d'actualiser la page".
